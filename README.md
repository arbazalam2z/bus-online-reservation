# bus-online-reservation
Bus Online Reservation

Sample Bus Online Reservation, in this case I using Bandung Express.

##Features:
1. Ticket Booking
2. Seat Layout Choose
3. Confirmation
4. Download Invoice of Booking

## Usage
Open project using Android Studio
